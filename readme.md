В некоторых папках есть картинки роботов которые помогут  опознать какой код идет куда


Ultrasound - пример как измерять расстояние с ультразвуком HC-SR04 

turning_head - пример серво-мотора, просто мотает головой

Reading_color_sensor - пример как считывать цвет с TCS230

_3_sensors_2_motors - код для робота-жука с 3мя датчиками препядствий. Но этот вид ИК датчиков с общей платой на 4 датчика работает хуже чем отдельные синие ИК датчики

distance_led_matrix - отображает расстояние на светодиодной матрице и ездит от препядствия

backwards_forwards - один ИК и один датчик касания, ездит назад-вперед

ESP_AP - код для WIFI-управляемой машины. Имя сети EvoAllTerrain-1D5, Пароль manchester, адрес 192.168.1.1
